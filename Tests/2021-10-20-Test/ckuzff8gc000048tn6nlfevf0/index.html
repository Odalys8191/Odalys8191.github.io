# 夏令营

# 游戏

nnn 对人组成大小为 2n2n2n 的环，黑白染色构造使得每对人颜色不同，相邻三人颜色不完全相同。

1→2,2→31\to 2, 2\to 31→2,2→3 这样连边，每对人连边，黑白染色。

这里连边的含义表示颜色不同，那么要求我们证明这玩意是个二分图。

然后这样连边度数都为偶数，是不会存在奇环的，那他就是二分图。

# 字符串

给每件物品一个 010101 编码，使得不存在一个编码与另一个编码有前缀关系，最小化 ∑ai×li\sum a_i \times l_i∑ai​×li​ 。

不存在编码前缀关系就对应着在 trietrietrie 树上无祖先关系

那就 trietrietrie 树上 dpdpdp ，设 dpi,j,kdp_{i,j,k}dpi,j,k​ 表示 trietrietrie 树上到了第 iii 层，已经放了 jjj 个物品，同层还能放 kkk 个的最小代价

推荐一种精妙的写法。
